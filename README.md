
Projet : Gestion des matériels universitaires – Université Nangui Abrogoua
Description

Ce projet consiste à développer une application web pour la gestion des matériels et infrastructures 
de l’Université Nangui Abrogoua. L’objectif est de faciliter le suivi et l’organisation des ressources 
disponibles dans l’université, telles que :

    Les salles de cours et de soutenance

    Les matériels informatiques

    Les équipements de la bibliothèque

     Autres matériels et infrastructures nécessaires à la vie universitaire

    Fonctionnalités principales

    Gestion des salles (création, modification, réservation)

    Gestion des matériels informatiques (inventaire, état, attribution)

    Suivi des salles de soutenance et des équipements associés

Interface simple et intuitive pour les administrateurs et utilisateurs

Technologies utilisées

Frontend : HTML, CSS, JavaScript

Backend : Django (Python)

Base de données : MySQL / SQLite

Objectif

Cette application vise à améliorer l’organisation et la gestion des ressources de l’Université Nangui Abrogoua, 
à réduire les pertes ou oublis de matériels, et à simplifier le travail des responsables administratifs et techniques.
